Detailed Description

Une Epreuve est une compétition associée à un sport

Les épreuves sont associées à un seul sport. Deux épreuves d'un même sport ne peuvent avoir le même nom; mais le même nom peut être utilisé pour 2 sports.

Exemples d'épreuves:

  • pour Atlhétisme
    • Cadet féminin 100m
  • pour Baseball
    • Tournoi masculin
